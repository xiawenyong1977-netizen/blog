# 快速开始指南（5分钟部署）

## 🚀 立即部署

### 步骤1：创建GitHub仓库

1. 登录GitHub：https://github.com
2. 点击右上角 **+** → **New repository**
3. 仓库名称：`xintuxiangce-blog`
4. 设置为 **Public**（GitHub Pages免费版需要）
5. **不要**勾选"Initialize this repository with a README"
6. 点击 **Create repository**

### 步骤2：上传博客文件

在PowerShell中执行：

```powershell
# 进入blog目录
cd d:\xintuxiangce\blog

# 初始化Git仓库
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit: GitHub Pages blog"

# 添加远程仓库（替换YOUR_USERNAME为您的GitHub用户名）
git remote add origin https://github.com/YOUR_USERNAME/xintuxiangce-blog.git

# 推送到GitHub
git branch -M main
git push -u origin main
```

**注意**：将 `YOUR_USERNAME` 替换为您的GitHub用户名（例如：`xiawenyong1977-netizen`）

### 步骤3：启用GitHub Pages

1. 进入GitHub仓库页面：`https://github.com/YOUR_USERNAME/xintuxiangce-blog`
2. 点击 **Settings**（设置）
3. 左侧菜单找到 **Pages**
4. 在 **Source** 部分：
   - 选择 **Deploy from a branch**
   - Branch选择 **main**
   - Folder选择 **/ (root)**
5. 点击 **Save**（保存）

### 步骤4：访问博客

等待1-2分钟，然后访问：

- **默认地址**：`https://YOUR_USERNAME.github.io/xintuxiangce-blog`
- 例如：`https://xiawenyong1977-netizen.github.io/xintuxiangce-blog`

## 🌐 配置自定义域名（可选）

### 步骤1：配置DNS

在您的域名DNS管理中添加CNAME记录：

```
类型: CNAME
名称: blog
值: YOUR_USERNAME.github.io
TTL: 3600
```

例如，如果您的GitHub用户名是 `xiawenyong1977-netizen`，则：
```
类型: CNAME
名称: blog
值: xiawenyong1977-netizen.github.io
```

### 步骤2：在GitHub中配置

1. 确保 `blog/CNAME` 文件存在，内容为：`blog.xintuxiangce.top`
2. 如果还没有，执行：
   ```powershell
   cd d:\xintuxiangce\blog
   echo "blog.xintuxiangce.top" > CNAME
   git add CNAME
   git commit -m "Add CNAME for custom domain"
   git push
   ```

3. 在GitHub仓库设置中：
   - Settings → Pages → Custom domain
   - 输入：`blog.xintuxiangce.top`
   - 勾选 **Enforce HTTPS**

### 步骤3：等待生效

DNS记录通常需要几分钟到几小时生效。

## 📝 添加新文章

### 创建文章文件

在 `blog/_posts` 目录中创建新文件，文件名格式：`YYYY-MM-DD-文章标题.md`

```powershell
# 示例：创建新文章
cd d:\xintuxiangce\blog\_posts
# 创建文件：2025-12-16-AI智能分类技术详解.md
```

### 文章模板

```markdown
---
layout: post
title: "文章标题"
date: 2025-12-16
categories: [分类1, 分类2]
tags: [标签1, 标签2]
author: 芯图团队
---

## 文章内容

这里是文章正文...

## 添加指向主网站的链接

了解更多，请访问[芯图相册官网](https://www.xintuxiangce.top) - AI智能照片分类工具。

<!-- 这个链接是dofollow，会传递SEO权重！ -->
```

### 发布文章

```powershell
cd d:\xintuxiangce\blog

# 添加文件
git add _posts/2025-12-16-AI智能分类技术详解.md

# 提交
git commit -m "Add: AI智能分类技术详解"

# 推送
git push
```

GitHub Actions会自动构建和部署，通常几分钟内就能看到更新。

## 🆘 遇到问题？

查看详细文档：`部署指南.md`

常见问题：
1. **博客无法访问**：检查GitHub Pages是否已启用，仓库是否为Public
2. **自定义域名不工作**：检查DNS记录和CNAME文件
3. **文章不显示**：检查文件名格式和Front Matter
