## 🚀 下一步操作

### 1. 创建GitHub仓库并推送代码（5分钟）

```powershell
# 进入blog目录
cd d:\xintuxiangce\blog

# 初始化Git仓库
git init

# 添加所有文件
git add .

# 提交
git commit -m "Initial commit: GitHub Pages blog"

# 添加远程仓库（替换YOUR_USERNAME为您的GitHub用户名）
# 例如：git remote add origin https://github.com/xiawenyong1977-netizen/xintuxiangce-blog.git
git remote add origin https://github.com/YOUR_USERNAME/xintuxiangce-blog.git

# 推送到GitHub
git branch -M main
git push -u origin main
```

**重要**：在GitHub上先创建仓库 `xintuxiangce-blog`（Public），然后再执行上述命令。

### 2. 启用GitHub Pages（2分钟）

1. 进入GitHub仓库：`https://github.com/YOUR_USERNAME/xintuxiangce-blog`
2. Settings → Pages
3. Source选择：**Deploy from a branch**
4. Branch选择：**main**
5. Folder选择：**/ (root)**
6. 点击 **Save**

### 3. 访问博客

等待1-2分钟，访问：
- 默认地址：`https://YOUR_USERNAME.github.io/xintuxiangce-blog`
- 自定义域名：`https://blog.xintuxiangce.top`（需要配置DNS）

## 🔗 外链策略

### 在文章中添加dofollow链接

所有在博客文章中的链接默认都是**dofollow**，会传递SEO权重：

```markdown
<!-- 好的链接方式：使用关键词作为锚文本 -->
[照片分类工具](https://www.xintuxiangce.top) - AI智能分类，90%+准确率。

[芯图相册](https://www.xintuxiangce.top)支持8大分类维度。

<!-- 避免的方式 -->
点击[这里](https://www.xintuxiangce.top)了解更多。
```

### 链接位置建议

- ✅ **正文中**：权重最高
- ✅ **相关文章推荐**：权重较高
- ⚠️ **侧边栏**：权重较低
- ❌ **页脚**：权重最低

### 每篇文章的链接数量

- ✅ **2-3个链接**：自然、不过度
- ❌ **避免**：10+个链接（会被视为垃圾链接）

## 📊 SEO优化建议

### 1. 文章标题
- ✅ 包含关键词：如"照片分类"、"AI智能分类"
- ✅ 长度控制在50-60字符
- ✅ 吸引点击：使用数字、疑问句等

### 2. 文章描述
在Front Matter中添加description：

```yaml
---
title: "照片分类完全指南"
description: "详细介绍8种照片分类方法，帮助您高效整理照片。芯图相册支持AI智能分类，90%+准确率。"
---
```

### 3. 关键词使用
- ✅ 主关键词在标题中出现1次
- ✅ 主关键词在正文中自然出现8-12次
- ✅ 使用相关关键词和同义词

### 4. 内部链接
- ✅ 链接到主网站的相关页面
- ✅ 链接到博客中的其他相关文章
- ✅ 使用关键词作为锚文本
